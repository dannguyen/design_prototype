Search Tips

open.contractors returns different page types for each of the following types of queries:

  • company name
  • geographic location
  • agency name
  • contract PIID number
  • general queries

It is best to start with only one search term and narrow from there.

About the data

Data on this site was taken from USAspending.gov, the American government’s website for open and unclassified contractor data.

This site does not include classified data.

open.contractors also aggregates from other government data sources including Congress’s membership database and the SEC’s EDGAR database.
